# Задания на практику для студентов НГТУ из ПМ-3x

Для работы с репозиторием рекомендуется использовать [Visual Studio Code](https://code.visualstudio.com/) и [devcontainers](https://code.visualstudio.com/docs/devcontainers/containers).

## Выполнение заданий
Для выполнения заданий, каждый студент должен создать в директории `students` поддиректорию со своей фамилией (маленькими латинскими буквами). Затем, для каждого задания, создать там класс с методами как в директории `students/test`.

## Lint & Test
1. Для запуска всех линтеров: **Run Build Task** (Ctrl+Shift+B). 
2. Для запуска всех тестов: **Run Test Task**.
